# Version 0.0.1

第一个发布版本.

- 支持一下字段:

    - StringField
    - NumberField
    - IntegerField
    - FloatField
    - BoolField
    - UUIDField
    - MD5Field
    - SHAField
    - EmailField
    - IPAddressField
    - URLField
    - EnumField
    - DictField
    - ListField
    - TimestampField
    - DatetimeField
    - DateField

- 支持自定义字段级字段验证

- 支持自定义全局数据验证
